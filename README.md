# My Spring Boot Project Task 1

This is a simple web app made with Spring Boot that displays a greeting page with an image in PNG format of the Vistula logo.

## How to run
1. Open the project in IntelliJ  
2. Run `FirstProjectJavaSpringApplication`  
3. Visit [http://localhost:8080/greeting](http://localhost:8080/greeting)
